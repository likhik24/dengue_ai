Open the folder dengue_ai and run the command in command line
python train.py
or 
python3 train.py

This command gives the r2 value and mean absolute error of all the models we trained and tested

Now run
python test.py
or 
python3 test.py

this command generates a file named submission_output.csv which generates the total cases column for the dengue_features_test.csv dataset
We uploaded this csv in driven data competetion link below and got mean absolute error value of 25.9

https://www.drivendata.org/competitions/44/dengai-predicting-disease-spread/
